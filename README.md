## Run Setup

```bash
# Para clonar o projeto
https://github.com/FelsenOficial/Schaf.git

# Entrar no diretório do projeto
cd Schaf

# Instalar as dependências
npm i

# Iniciar a aplicação
npm run dev

# Iniciar migration
npm run typeorm migration:create -- -n nomeDaMigration

# Upar Migration
npm run typeorm migration:run
```
